Pitcher Whiff Skill

Leaderboard Playoffs Methodology

Methodology & model details

How whiff skill is calculated and what data is included.

Metric definition

Whiff skill = actual whiff rate - expected whiff rate. Positive values indicate a pitcher generated more whiffs than expected given pitch context.

Modeling approach

  • Expected whiff rates are generated using an XGBoost model with isotonic calibration.
  • The model is trained on regular-season data and then applied to regular-season and postseason samples for comparison.
  • Confidence/qualification thresholds are based on minimum pitch counts shown in the metadata file.

Data source

Baseball Savant (Statcast).
